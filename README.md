# magento2-checkout-newsletter-subscription
The module adds a newsletter subscription checkbox to the default checkout sidebar for Luma.

### Installation

`vendic/magento2-checkout-newsletter-subscription`

### Admin settings 

Stores > Configurations > Vendic > Checkout Newsletter Subscription
